Wordpress Website Dragon Slayers' Depot

Visit Website

Dragon Slayers' Depot is a mock ecommerce site made with the WordPress CMS. This mock business sells fantasy adventuring gear such as enchanted swords, potions, and magical staffs.

  • Responsive display - adapts to mobile, tablet, and desktop screens
  • Implements the WooCommerce ecommerce plugin to create dynamic product pages and cart/checkout functionality
  • Static pages created using the Elementor visual builder
  • Uses pre-made theme
  • Uses WebP image format for greater page loading speed
  • Includes a subscription capability for receiving newsletters
  • Secured with HTTPS

Wildfire Spread Simulation

Visit Github Page

The Wildfire simulation is a program that calculates fire spread behavior based on varying environmental factors and outputs the result in the form of animation or statistics. Made as a personal project.

  • Independently developed a continuous agent-based model for simulating wildfires
  • Collected spread rate, area, and reaction intensity statistics
  • Calculations based on changes in wind, slope, temperature, fuel type, and moisture level
  • Animation used to study the shape and speed at which land agents/units transfered between the initial, burning, and burnt states
  • Implemented with Object Oriented Design in MATLAB
